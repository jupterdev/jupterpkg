# Jupter Pkg

Uma package para node.js criada recentemente!

# Versão

v1.0.0

# Instalação

`npm i jupterpkg`

# Como utilizar

```
import { jupterpkg } from 'jupterpkg'

jupterpkg({
    shadow_type: 'soft',
    padding: false
});
```

## Opções

jupterpkg suporta 2 opções, na quais são:

- _shadow_type_ - _hard | soft_ (O padrão e soft)
- _padding_ - _boolean_ (o padrão e false)
